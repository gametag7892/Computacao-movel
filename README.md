# 📱 Estudos de Computação Móvel - Flutter

Este repositório contém os exercícios práticos desenvolvidos durante as aulas de **Computação Móvel** no curso de **Sistemas de Informação** na **ESAMC**, sob orientação do **Prof. Yassushi Okada**.

O foco do curso é o domínio do framework Flutter e da linguagem Dart para a criação de aplicações multiplataforma modernas, performáticas e responsivas.

---

## 🚀 Roteiros de Aprendizado

### 📝 Roteiro 1: Conceitos e Estrutura Inicial
* **Fundamentos Mobile**: Estudo de interfaces touch, performance e responsividade.
* **Flutter Engine**: Entendimento do funcionamento da engine **Skia** para renderização de componentes.
* **Configuração**: Setup do SDK, Git e integração com IDEs (VS Code/Android Studio).
* **Widgets**: Diferenciação prática entre `StatelessWidget` e `StatefulWidget`.

### 🗺️ Roteiro 2: Navegação e Fluxo de Dados
* **Pilha de Telas**: Gerenciamento de rotas com `Navigator` (`push` e `pop`).
* **Comunicação**: Passagem de parâmetros via construtor entre diferentes telas.
* **Assincronismo**: Implementação de `async`, `await` e `Future` para lidar com respostas de telas anteriores.

### 🏗️ Roteiro 3: Organização e Listas
* **Arquitetura**: Organização do projeto seguindo o padrão de pastas `lib/screens/` e `lib/widgets/`.
* **Componentização**: Criação de widgets customizados e reutilizáveis (Ex: `CustomButton`).
* **Listas Dinâmicas**: Uso eficiente de `ListView.builder` para renderização sob demanda.

### 🔄 Roteiro 4: Ciclo de Vida e APIs
* **Lifecycle**: Manipulação dos métodos `initState()` e `dispose()` para gestão de memória e recursos.
* **Gerenciamento de Estado**: Uso do `setState()` para atualização reativa da UI.
* **Integração PokéAPI**: Consumo de APIs REST com o pacote `http`.
* **Models**: Serialização de dados JSON para objetos Dart através de métodos `fromJson`.

---

## 🛠️ Tecnologias e Dependências

* **Linguagem:** Dart
* **Framework:** Flutter
* **Pacotes Principais:**
  * `http`: Requisições assíncronas para APIs.
  * `audioplayers`: Execução de efeitos sonoros no app.
